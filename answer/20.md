### 解答
```
$ for i in $(cd /usr;echo *); do echo $i ; done
```
### 別解
```
別解1（上田）$ for f in /usr/* ; do echo ${f##*/} ; done
別解2（上田）$ a=(/usr/*); echo -e ${a[@]/\/usr\//\\n}
```

### 読者別解

```
別解3 $ (cd /usr; set -- *; IFS=$'\n'; echo "$*")
```
